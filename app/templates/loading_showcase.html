Exemples de code

Glassmorphism CSS

.loading-content { /* Glassmorphism effect */ background: rgba(255, 255, 255, 0.95); backdrop-filter: blur(20px) saturate(180%); -webkit-backdrop-filter: blur(20px) saturate(180%); /* Border with gradient */ border: 1px solid rgba(255, 255, 255, 0.6); /* Elegant shadow with multiple layers */ box-shadow: 0 8px 32px rgba(0, 0, 0, 0.08), 0 2px 8px rgba(0, 0, 0, 0.04), inset 0 1px 0 rgba(255, 255, 255, 0.8); }

Multi-ring Spinner

/* Outer ring */ .loading-spinner::before { content: ''; width: 80px; height: 80px; border: 3px solid transparent; border-top-color: #3498db; border-right-color: #3498db; animation: spinFast 1.2s cubic-bezier(0.68, -0.55, 0.265, 1.55) infinite; filter: drop-shadow(0 0 8px rgba(52, 152, 219, 0.4)); } /* Inner ring */ .loading-spinner::after { content: ''; width: 56px; height: 56px; border: 3px solid transparent; border-bottom-color: #60a5fa; border-left-color: #60a5fa; animation: spinSlow 1.8s cubic-bezier(0.68, -0.55, 0.265, 1.55) infinite reverse; } /* Pulsing center dot */ .loading-spinner { background: radial-gradient(circle, #3498db 0%, #60a5fa 100%); width: 16px; height: 16px; animation: pulse 2s cubic-bezier(0.4, 0, 0.6, 1) infinite; }

Utilisation JavaScript

// Créer une instance const loadingOverlay = new LoadingOverlay(); // Afficher l'overlay loadingOverlay.show('Chargement...', 'Analyse du fichier PCAP'); // Mettre à jour le message loadingOverlay.update('Étape 2/3', 'Traitement des paquets...'); // Masquer l'overlay loadingOverlay.hide();
